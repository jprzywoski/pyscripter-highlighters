Welcome to the PyScripter highlighters repository.


Date: Tue Jun 03 2015

This repository contains config files containing various code highlighting color schemes.


Links:
<PyScripter>

Usage:

- fire up PyScripter
- click Tools > Options > Import/Export > Import Highlighters
- choose the file you want to import


Note:

Importing a new highlighter overwrites the one you are currently using.
